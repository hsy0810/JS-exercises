```sh
console.log(a, b, c);
# 出力値：0,1,0

console.log(a, b, e);
# 出力値：1,1,0
```

- 説明
```sh
# a++ は a の値（0）を使用した後に a をインクリメントします。その結果、c にはインクリメント前の a の値である 0 が代入されます。
# bはaをインクリメント後の値を代入するため、1となります。
const c = a++ +b;

# a++ は a の値（現在の a の値は 1）を使用した後に a をインクリメントします。そして、その評価値である 1 が e に代入されます。
# b; はただの独立した式であり、e への代入には影響を与えません。
const e = a++; b;
```
